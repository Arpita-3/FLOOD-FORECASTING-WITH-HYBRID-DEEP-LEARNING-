🌊 Flood Prediction using Hybrid Deep Learning Models Floods cause major socioeconomic damage, so accurate forecasting is critical. This project builds and compares advanced deep learning models to predict river discharge and improve flood preparedness. Using 6 years of real meteorological and hydrological data from the Tel River Basin, multiple models were developed and evaluated, including LSTM, BiGRU, Transformer, and Temporal Fusion Transformer (TFT). The Temporal Fusion Transformer (TFT) achieved the best performance with R² > 0.95, showing strong ability to capture long-term temporal patterns and nonlinear behavior in river systems. 🔍 Key Highlights Multi-model deep learning comparison Hybrid Transformer-based architectures Time series forecasting with real-world data Metrics: R², RMSE, MAE, NSE Attention-based modeling for better interpretability 🧠 Tech Stack Python · Tensorflow · Transformers · Time Series · Deep Learning
